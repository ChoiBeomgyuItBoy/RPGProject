  - Project Dragon is a Fantasy RPG, Action, Point & Click style game.
  - Architecture Modules:
    - Point & Click Movement
    - Enemy AI (FSM simulation)
    - Combat 
    - Cinematics
    - Inventories
    - Dialogues
    - Quests
    - Shops
    - Abilities
